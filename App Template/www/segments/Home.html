Cette page est entièrement définie dans un fichier segment .html

Les fichiers de segment peuvent être de 3 types

  • Template HTML avec extention .html, le nom du fichier est égelement partie du nom de la section dans la page application: section_X (X = nom fichier)
  • Fichier javascript avec extention .js
  • Fichier de style avec extention .css, le nom du fichier est égelement partie du nom de la section dans la page application: style_X (X = nom fichier)
